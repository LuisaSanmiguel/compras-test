Clonar


$ git clone https://github.com/LuisaSanmiguel/compras-test.git

Ingresar a la carpeta de proyecto
cd compras-test

$composer install

copiar  .env-example, colocarle nombre .env y configurar
$php artisan migrate

$npm run dev
